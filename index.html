We have two categories of simulators

Programming Simulators
(self explanatory simulators)

This type of simulators are having mostly C programming codes and they are self explanatory e.g code are executing line by line and you can understand them easily . these are the mock examples which will make you learn about programming concepts and by this website you will have an idea that how simulator is created and how code runs.

Non self explanatory

This type of simulators are just showing their functionality ,the code is not running line by line but after watching these simulators you will get the idea of programming and frontEnd web development using HTML5, CSS and js.

Most of the students have created simulators based on programs and many others have created to demonstrate other functionalities and action like something intresting is happening on clicking some button etc.
